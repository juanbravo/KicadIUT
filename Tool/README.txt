KicadLibrarian est un outil de gestion pratique (compatible avec Wine pour ceux sous Linux):
- déplacement d'empreintes 
- création de librairies
- modif de pastilles
- visu d'empreintes 
etc..............

Vous avez 2 modes de travail: View>schematic (pour les symboles graphiques) ou View>Footprint (pour les empreintes)
N'oubliez pas d'aller dans Préferences>SearchPath pour définir le chemin des librairies
